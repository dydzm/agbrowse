№ 01 — WEB-AI SESSIONS

Web AI sessions.
Real browser.
One CLI.

agbrowse runs ChatGPT, Gemini, and Grok web sessions through real Chrome tabs — every answer backed by an observable page, not a hidden API.

Developer docs →
# send once, poll from any shell
$ agbrowse web-ai send \
  --vendor chatgpt \
  --model pro \
  --prompt "Cite sources."
session s_9f2e41 created
$ agbrowse web-ai poll \
  --session s_9f2e41
→ result.md (2,914 words)
