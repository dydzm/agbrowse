레퍼런스

설정은 로컬 환경변수 중심입니다.

이름기본값역할
BROWSER_AGENT_HOME~/.browser-agentBrowser state, session, lease, trace 저장 위치.
CDP_PORT9222Chrome DevTools Protocol port.
AGBROWSE_JSON_ERRORS미설정1이면 --json 여부와 무관하게 JSON error envelope를 강제합니다.
AGBROWSE_UPDATE_CHECKCI 밖에서 활성0이면 npm 업데이트 알림 숨김, 1이면 강제 확인.
AGBROWSE_UPDATE_CHECK_TTL24hnpm 최신 버전 확인 캐시 TTL.
AGBROWSE_UPDATE_CHECK_LATEST미설정테스트/진단용 최신 버전 override.
AGBROWSE_WEB_AI_AUTO_START활성0이면 Chrome이 없을 때 자동 시작하지 않고 실패합니다.
AGBROWSE_PROVIDER_POOL_MAX_PER_KEY미설정Provider warm tab pool 크기 override.
CHROME_HEADLESS미설정1이면 headless 시작.
CHROME_NO_SANDBOX미설정Docker/CI에서 필요 시에만 1로 설정.
CHROME_BINARY_PATH자동 감지Chrome 실행 파일 경로.
BROWSER_SCRIPT번들 포함커스텀 browser script 경로 (vision-click용).

Local-first 경계

agbrowse는 동작 상태를 로컬에 저장합니다. Hosted model routing, provider API-key auth, cloud browser execution은 추가하지 않습니다.

릴리즈 게이트첫 실행
로컬 browser home, trace, download, generated code artifact는 commit하지 않습니다.
